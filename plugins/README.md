## Installation

1. Download the `.plugin.js` file
2. Move it to your BetterDiscord plugins folder:
   - Windows: `%appdata%/BetterDiscord/plugins/`
3. Enable the plugin in BetterDiscord settings
4. Make sure Solari is running
