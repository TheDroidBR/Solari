How Solari Works

🤖

Auto-Detect

Solari automatically detects which game or app you are using and updates your status.

🆔

App Profiles

Each preset can have its own Client ID. You can appear as "Playing Visual Studio" and "Playing GTA" correctly without changing settings manually.

🔊

SoundBoard

Play sounds directly into your Discord voice channel. Requires VB-Cable driver (can be set up later).

⚙️ Quick Settings

Configure how Solari behaves on your system.

🚀

Start with Windows

Launch Solari automatically when your PC boots.

📥

Start Minimized

Open in the system tray instead of showing the window.

🔽

Minimize on Close

Closing the window keeps Solari running in the tray.

🔄

Auto-Check Updates

Automatically check for new Solari versions on startup.

Identidade Base (Client ID)

O Client ID é o código que determina qual nome de "jogo" aparecerá no seu status do Discord.

O Solari exige pelo menos 1 Client ID padrão como fallback. Escolha uma opção pública abaixo ou configure um ID personalizado.

BetterDiscord

Detectando...

BetterDiscord is required to load client-side integrations into Discord.

💡 Como Ativar os Plugins

  1. Abra as Configurações do Discord (ícone de engrenagem ⚙️).
  2. Na barra lateral, acesse a aba Plugins na seção do BetterDiscord.
  3. Ative a chave (toggle) ao lado de cada plugin desejado.

⚠️ Nota: Este passo manual não é necessário se o plugin Solari Manager estiver instalado e ativado no BetterDiscord, pois ele gerencia os outros plugins automaticamente!
