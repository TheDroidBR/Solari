Driver de Áudio Necessário

Para usar o SoundBoard no Discord, você precisa instalar o VB-Audio Virtual Cable (grátis).

📋 Como Instalar:

  1. Clique no botão abaixo para baixar o VB-Cable
  2. Execute o instalador como Administrador
  3. Reinicie o computador
  4. Volte aqui e clique em "Verificar Driver"

  1. Abra Configurações → Voz e Vídeo
  2. Em "Dispositivo de Entrada", selecione "CABLE Output"
  3. Pronto!

How Solari Works

🤖

Auto-Detect

Solari automatically detects which game or app you are using and updates your status.

🆔

App Profiles

Each preset can have its own Client ID. You can appear as "Playing Visual Studio" and "Playing GTA" correctly without changing settings manually.

🔊

SoundBoard

Play sounds directly into your Discord voice channel. Requires VB-Cable driver (can be set up later).

⚙️ Quick Settings

Configure how Solari behaves on your system.

🚀

Start with Windows

Launch Solari automatically when your PC boots.

📥

Start Minimized

Open in the system tray instead of showing the window.

🔽

Minimize on Close

Closing the window keeps Solari running in the tray.

🔄

Auto-Check Updates

Automatically check for new Solari versions on startup.

🆔 Global Client ID

This is used when no specific profile is active.

Most users can skip this step. You can configure it later.
